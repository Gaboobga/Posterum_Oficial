# Posterum_Oficial
Posterum es una agencia enfocada en automatización de procesos y asistentes con IA. Este proyecto consiste en una landing page para presentar claramente la propuesta de valor, servicios y un llamado a la acción para agendar una demo/contacto, con un diseño minimalista y profesional.

La prioridad fue construir una página rápida, clara y orientada a conversión, cuidando jerarquía visual, mensajes simples y navegación intuitiva. Implementé una estructura escalable con HTML5, CSS3 (BEM) y JavaScript, además de control de versiones con Git/GitHub y publicación en GitHub Pages con dominio personalizado (DNS) para un despliegue en producción.

## Conclusiones principales

-Una estructura limpia + diseño responsivo mejora la lectura y la claridad del mensaje.

-Tener el sitio en producción con dominio propio facilita compartirlo y validarlo con clientes reales.

-Documentar y versionar con Git permite iterar rápido sin perder estabilidad.
